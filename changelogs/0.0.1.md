# Changelog 0.0.1

## [0.0.1] - 2026-06-14

### Fixed
- Fixed a fatal 500 error on case-sensitive file systems (production) by renaming the autoloader file `AutoLoader.php` to `Autoloader.php` (matching its PSR-4 class name `Autoloader`).
- Updated the require/use statements in `cron/gdpr_purge.php` and instantiated the Autoloader object correctly instead of calling `register()` statically.
- Relocated the installation check in `index.php` to the very top, before configuration imports. This avoids a 500 Internal Server Error when starting a fresh installation since configuration files (`app.php`) do not exist yet.
